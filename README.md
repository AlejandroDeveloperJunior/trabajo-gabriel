# Proyecto Web - Unidad 1

Sitio web desarrollado con PHP y Bootstrap como proyecto académico.

## Requisitos

- PHP 7.4 o superior
- Servidor web (Apache/Nginx) o `php -S localhost:8000`

## Cómo ejecutar

```bash
cd proyecto-web
php -S localhost:8000
```

Luego abre `http://localhost:8000` en tu navegador.

## Páginas incluidas

| Página | Ruta |
|--------|------|
| Inicio | `index.php` |
| Nosotros | `pages/about.php` |
| Servicios | `pages/services.php` |
| Contáctanos | `pages/contact.php` |
| Registrar | `pages/register.php` |
| Iniciar sesión | `pages/login.php` |
| Recuperar contraseña | `pages/recovery.php` |
| Buzón | `pages/inbox.php` |
| Ayuda | `pages/help.php` |
| Búsqueda | `pages/search.php` |
| Mapa del sitio | `pages/sitemap.php` |
| Error 404 | `404.php` |

## Funcionalidades

- Menú de navegación con ID del sitio
- Búsqueda interna del sitio
- Validación de datos: frontend (JavaScript) y backend (PHP)
- CAPTCHA humano en registro
- Chat flotante en todas las páginas
- Recuperación de contraseña
- Buzón de mensajes
- Página de error 404
- Mapa del sitio

## Tecnologías

- PHP (Orientado a Objetos)
- Bootstrap 5.3
- JavaScript vanilla
